# Heart Disease Prediction using Hybrid Ensemble Model

## Overview
This project predicts heart disease using advanced machine learning techniques and a hybrid ensemble model.

## Techniques Used
- Feature Selection: SelectKBest (ANOVA F-score)
- Data Scaling: StandardScaler
- Imbalance Handling: SMOTE

## Models Used
- K-Nearest Neighbors (KNN)
- Gradient Boosting
- XGBoost
- Extra Trees Classifier

## Ensemble Methods
- Voting Classifier
- Stacking Classifier (Meta: Logistic Regression)

## Result
Achieved improved accuracy using hybrid ensemble architecture.

## Technologies
Python, Scikit-learn, XGBoost, Pandas, NumPy
